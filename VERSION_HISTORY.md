## ✅ 已发布版本

### [V1.0.0] - 2026-06-27
**状态：** 首发版本
**APK 位置：** `versions/V1.0.0/`

#### 新增
- 猫咪管理（增删改查、照片、健康状态）
- 医疗记录（疫苗、驱虫、体检日历提醒）
- 收养流程（申请→初审→家访→审批→入住）
- Excel 数据导出（share_plus）
- 本地 Hive 数据库（离线优先架构）
- Flutter 动态字数统计
- Material 3 暖橙主题 + 猫爪图标
- 管理员/游客双权限模式
- 微信维护群二维码（设置页）
- Excel 导出本地存储

#### 界面
- 首页仪表盘（猫咪总数/待家访/待绝育/本月医疗）
- 猫咪列表（筛选、搜索、卡片展示）
- 猫咪详情（照片轮播、成长日记、医疗时间线）
- 医疗日历（中文本地化）
- 收养管理（全流程审批）
- 设置页（导出、清理、关于）
- 登录页（可跳过，直接浏览）

#### 技术
- Flutter 3.32.0
- Provider 状态管理
- GoRouter 路由
- Hive 本地存储
- API 服务层（占位，云端后端待部署）
